Reconhecimento Facial Marinho com Gemini

A Blue Vision apresenta o desenvolvimento de uma ferramenta de reconhecimento facial para a identificação e monitoramento da biodiversidade marinha. Essa ferramenta permite coletar dados essenciais para proteger os ecossistemas oceânicos.

Objetivo

Desenvolvimento de Ferramenta

Criar uma solução tecnológica avançada para identificar espécies marinhas a partir de imagens de drones.

Apoiar Conservação e Pesquisa

Fornecer dados precisos para subsidiar ações de conservação e avanços na pesquisa científica.
